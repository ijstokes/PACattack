IPython Notebook Turtle Module

The IPython notebook is an interactive environment where the user can write Python code within their browser, without needed Python installed in their computer. For this project we created a turtle module to be used in the IPython Notebook. This module that we created was meant to mimic the turtle module used in Python.

The Turtle module is a module used for drawing. The user controls the turtle with simple commands. The commands can manipulate the turtle so it moves, rotates, and while it's moving it can draw (or not draw, depending on whether the user has specified if the "pen" is up or down). If the pen is down and the turtle is moving, it will draw a line under it's path.

Uses for the Turtle Module

Using the turtle is simple, and the results of coding actions are immediately visible. This makes it a great way to introduce programming to both children and adults. The visual feedback of the drawing turtle makes programming less abstract for someone who is new to programming. Because of the quick visual feedback of the user's commands, and the IPython Notebook's cell format (allowing the user to see the code directly above their turtle), programming becomes a easier, not only to learn, but also to teach. A teacher can easily teach this to the entire class, and have their students work on it individually because the IPython Notebook is browser based. Users can draw a variety of things using the turtle module.

Using the Turtle Module

By importing the module the user can use code to create and manipulate the turtle. An example of commands you can use to draw with the turtle are:

import pacturtle 
T = pacturtle.pacturtle() 
T.speed(7) 
T.right(33) 
T.pensize(4) 
T.forward(70) 
T.left(55) 
T.pencolor("blue") 
T.backward(100)

Installation Instructions

For Mac:

Step 1: Open the terminal. This can be done by clicking the magnifying glass in the top right corner of your screen and typing 'terminal'. You can also find the terminal application in Applications.

Step 2: You will need to go to the directory where you want the IPython Notebook stored. (Hint: You can create a directory by typing mkdir nameofdirectory and then to move into that directory within the terminal type cd nameofdirectory).

Step 3: Type git clone https://github.com/macewanCMPT395/PACattack.git into the terminal. This will copy the github repository with the developmental IPython Notebook with the turtle module into your directory.

Step 4: In the terminal type cd pacturtle, this is the directory you will be running the notebook from.

Step 5: To run our developmental IPython application type python -m IPython in the terminal. This will open the IPython Notebook application in your default browser.

Step 6: Click the "new notebook" button in the top right corner. This will create new notebook where you can add and run python code.

Step 7: Run some turtle commands (an example can be found if you click the button with "i" at the top of your newly created notebook, or refer to Figure 1) and enjoy making fun pictures!

For Windows:

Step 1: Open the MS-Dos console, or OS console command prompt.

Step 2: You will need to go to the directory where you want this notebook stored. Change into that directory.

Step 3: Type git clone https://github.com/macewanCMPT395/PACattack.git This will copy the github repository with the developmental IPython Notebook with the turtle module into your directory.

Step 4: In the console type cd pacturtle, this is the directory you will be running the notebook from.

Step 5: To run our developmental IPython application type python.exe -m IPython notebook.

If python is not a recognized alias you can also type c:\python27\python.exe -m IPython notebook in the console. Change the directory accordingly to where you installed python.

This will open a IPython Notebook in your default browser.

Step 6: Click the "new notebook" button in the top right corner. This will create new notebook where you can add and run python code.

Step 7: Run some turtle commands (an example can be found if you click "i" or refer to Figure 1) and enjoy making fun pictures!
